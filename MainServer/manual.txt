## 파일 업로드 및 관리 시스템 API 매뉴얼

**프론트엔드 개발자를 위한 안내**

이 API는 음성 데이터를 업로드하고 관리하며, 개인화된 음성 모델을 생성하고 TTS(텍스트 음성 변환) 기능을 제공합니다. 

**1. 사용자 생성**

*   **요청 방식:** POST
*   **URL:** `/user`
*   **데이터:** 없음 (빈 요청)
*   **응답:** 
    *   성공: `{ "uuid": "사용자 고유 UUID" }`
    *   실패: 500 에러 (사용자 생성 및 저장 오류)

**2. 파일 업로드**

*   **요청 방식:** POST
*   **URL:** `/user/{user_uuid}/upload`
*   **데이터:** 
    *   `user_uuid`: 사용자 고유 UUID
    *   `file`: 업로드할 오디오 파일 (.ogg 형식만 지원)
*   **응답:** 
    *   성공: `{ "message": "파일 업로드 성공", "file_path": "파일 경로" }`
    *   실패:
        *   400 에러 (허용되지 않은 파일 형식)
        *   500 에러 (파일 업로드 오류)

**3. 파일 목록 조회**

*   **요청 방식:** GET
*   **URL:** `/user/{user_uuid}/files`
*   **데이터:** 
    *   `user_uuid`: 사용자 고유 UUID
*   **응답:** 
    *   성공: `{ "files": [ "파일 이름 1", "파일 이름 2", ... ] }`
    *   실패: 500 에러 (파일 목록 조회 오류)

**4. 파일 삭제**

*   **요청 방식:** DELETE
*   **URL:** `/user/{user_uuid}/raw/{filename}`
*   **데이터:** 
    *   `user_uuid`: 사용자 고유 UUID
    *   `filename`: 삭제할 파일 이름
*   **응답:** 
    *   성공: `{ "message": "파일 삭제 성공" }`
    *   실패:
        *   404 에러 (파일을 찾을 수 없음)
        *   500 에러 (파일 삭제 오류)

**5. 파일 다운로드**

*   **요청 방식:** GET
*   **URL:** `/user/{user_uuid}/raw/{filename}`
*   **데이터:** 
    *   `user_uuid`: 사용자 고유 UUID
    *   `filename`: 다운로드할 파일 이름
*   **응답:** 
    *   성공: 다운로드할 WAV 파일
    *   실패:
        *   404 에러 (파일을 찾을 수 없음)
        *   500 에러 (파일 다운로드 오류)

**6. 모델 생성 요청**

*   **요청 방식:** POST
*   **URL:** `/user/{user_uuid}/models`
*   **데이터:** 
    *   `user_uuid`: 사용자 고유 UUID
    *   `sample_rate`: 샘플링 레이트 (기본값: 40000)
    *   `cpu_cores`: CPU 코어 수 (기본값: 4)
*   **응답:** 
    *   성공: `{ "message": "모델 생성 작업이 요청되었습니다.", "model_id": "모델 ID" }`
    *   실패: 500 에러 (모델 생성 요청 오류)

**7. TTS 합성 요청**

*   **요청 방식:** POST
*   **URL:** `/user/{user_uuid}/tts`
*   **데이터:** 
    *   `user_uuid`: 사용자 고유 UUID
    *   `tts_text`: TTS 합성에 사용할 텍스트
    *   `pitch`: 음성 높낮이 (기본값: 3)
*   **응답:** 
    *   성공: TTS 합성 결과 WAV 파일
    *   실패: 500 에러 (TTS 요청 오류)

**8. 사용자 삭제**

*   **요청 방식:** DELETE
*   **URL:** `/user/{user_uuid}`
*   **데이터:** 
    *   `user_uuid`: 사용자 고유 UUID
*   **응답:** 
    *   성공: `{ "message": "사용자 및 파일 삭제 성공" }`
    *   실패:
        *   404 에러 (사용자를 찾을 수 없음)
        *   500 에러 (사용자 삭제 오류)

**참고:**

*   모델 서버는 별도의 API를 제공하며, 메인 서버는 모델 서버 API와 통신하여 파일 업로드, 모델 생성, TTS 합성 요청을 처리합니다.
*   API는 개발 중에 변경될 수 있습니다. 최신 API 문서를 참조하세요.

**이 매뉴얼을 통해 프론트엔드 개발자는 파일 업로드 및 관리 시스템 API를 사용하여 사용자 계정을 생성, 파일을 업로드하고 관리하며, 개인화된 음성 모델을 생성하고 사용할 수 있습니다. **
